# **App Name**: City Unveiler

## Core Features:

- Interactive Map: Interactive map displaying undiscovered and discovered areas, fog-of-war style.
- Map Unveiling: RPG style reveal of the map with user movement.
- Route Tracking: GPS tracking of user's walking and biking routes.
- Distance Calculation: Distance Calculation to quantify how much walking/biking user did.
- Virtual Pet: A pet that evolves and levels up based on the distance and area covered during exploration.
- AI Guided Quiz: AI-powered tool which intelligently queries the user on points of interest as they reveal sections of the map.

## Style Guidelines:

- Background color: Dark background (#121212) to create contrast and depth, consistent with a dark interface.
- Primary color: Vibrant Green (#32CD32) as the main accent color to highlight interactive elements and progress indicators, representing exploration and discovery.
- Accent color: Muted Gold (#B8860B) for secondary highlights, indicating achievements, milestones, or special points of interest on the map.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, combined with 'Inter' (sans-serif) for body text, providing a modern, tech-inspired aesthetic with good readability.
- Custom icons for different points of interest, challenges, and rewards, designed in a modern style with a touch of playfulness.
- Subtle animations to give feedback to user interactions such as GPS data updates and map reveals.
- Intuitive map display with clearly labeled interactive zones and game stats.
